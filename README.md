# Window Organizer

GNOME Shell extension that moves each newly created normal window to the monitor currently under the mouse cursor.

## Install (local)

1. Copy `window-organizer/` to:
   `~/.local/share/gnome-shell/extensions/window-organizer@david-grieser.de/`
2. Restart GNOME Shell (or log out/in).
3. Enable the extension:
   `gnome-extensions enable window-organizer@david-grieser.de`
